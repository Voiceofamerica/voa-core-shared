# voa-core-shared

> Core shared code between server & mobile apps

## Shared Code

* GraphQL Schemas
* TypeScript interfaces

## Develop

* `npm install`
* Install recommend VS Code Extensions

## Publish

* `npm run build`
* **Commit** resulting changes
* `npm version minor`
* `npm publish`
* `git push --follow-tags`

### Dangerous

* `push-schema` auto commits and publishes
